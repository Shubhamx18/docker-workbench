## 🚀 About This Repository

Welcome to my **Docker learning journey for DevOps**.

This repository is a structured, hands-on roadmap where I learn Docker from the ground up — not just theory, but real commands, real setups, and real-world use cases used in DevOps and cloud environments.

Every topic here is documented as I practice it, making this repository a **live proof of my Docker skills**.

## 📚 Learning Roadmap

| Day | Topic | Description |
|-----|------|-------------|
| **Day 1** | What is Docker? | Definition, problems Docker solves, and how Docker works internally |
| **Day 2** | Containers & Docker Architecture | What containers are + detailed Docker architecture |
| **Day 3** | Docker vs Virtual Machines | Key differences + Docker core components |
| **Day 4** | Dockerfile & Docker Registry | Writing Dockerfiles and understanding image registries |
| **Day 5** | Running & Managing Containers | Detached mode, multiple containers, lifecycle management |
| **Day 6** | Image Management | Modifying images, rebuilding, base images, interactive mode |
| **Day 7** | Docker Hub & Image Sharing | Tagging, pushing, pulling, using images remotely |
| **Day 8** | Volumes & `.dockerignore` | Persistent storage, bind mounts, and ignoring files in builds |
| **Day 9** | Container Communication | Container ↔ container & container ↔ local DB (Python + MySQL example) |
| **Day 10** | Docker Networking | Bridge, host, custom networks, and communication rules |
| **Day 11** | Docker Compose & YAML | Compose files, YAML syntax, multi-container setup, volumes & ports |
| **Day 12** | Docker CLI Commands | All essential Docker commands for images, containers, logs, networks & cleanup |
