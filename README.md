## 🚀 About This Repository

This repository documents my **hands-on journey of mastering Docker for DevOps**.  
It contains practical commands, real examples, and mini-projects that show how Docker is used in modern deployment workflows.

Instead of just theory, this repo focuses on **building, running, and managing containers like a DevOps Engineer**.
